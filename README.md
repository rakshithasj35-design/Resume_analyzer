\# AI Resume Skill Gap Analyzer



This project analyzes a resume and compares it with a job description to find the skill match percentage and missing skills.



\## Features

\- Upload Resume PDF

\- Paste Job Description

\- Skill Match Percentage

\- Missing Skills Detection

\- Graph Visualization

\- Downloadable PDF Report Card



\## Tech Stack

\- Python

\- Streamlit

\- PyPDF2

\- Matplotlib

\- ReportLab



\## How to Run

1\. Install requirements

pip install -r requirements.txt



2\. Run the app

streamlit run app.py
